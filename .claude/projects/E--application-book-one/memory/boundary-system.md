# 境界体系

## 九级境界顺序

**正确顺序**：凡境 → 灵境 → 地境 → 天境 → 皇境 → 尊境 → 帝境 → 道境 → 混沌境

每级分四个阶段：初期、中期、后期、巅峰。

**关键纠正**：天境之后是皇境，不是地境！地境在灵境之后、天境之前。

## How to apply

创作时必须严格遵循九级境界顺序：

1. **不能跳境界**：必须依次通过凡→灵→地→天→皇→尊→帝→道→混沌
2. **不能逆转顺序**：天境后是皇境，不是地境
3. **每个大境界有四个小境界**：初期→中期→后期→巅峰
4. **创作前核对**：检查当前境界和下一境界是否正确衔接
