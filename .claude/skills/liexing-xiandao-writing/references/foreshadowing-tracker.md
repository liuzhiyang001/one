## 伏笔记录格式

每条伏笔记录包含以下字段：

| 字段 | 说明 |
|------|------|
| **编号** | 伏笔唯一编号，格式：VP-XXX |
| **类型** | 身世/世界/势力/物品/伙伴/其他 |
| **埋下章节** | 伏笔首次出现的章节 |
| **伏笔内容** | 具体埋下的线索或暗示 |
| **预期揭示** | 计划在哪一卷/章节揭示 |
| **揭示方式** | 通过什么方式揭示（对话/回忆/场景/战斗等） |
| **状态** | 已埋下/待埋下/已揭示/已闭环 |
