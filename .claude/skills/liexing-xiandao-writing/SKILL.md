# 《裂星衍道》写作指导

本skill用于创作百万字玄幻小说《裂星衍道》，融合辰东与我吃西红柿两位顶级网文作家的风格精髓。

## 写作风格融合指南

### 辰东风格要素

辰东的核心优势在于**宏大世界观构建**与**悬疑铺垫能力**：

1. **神秘感营造**
   - 每个遗迹、神器背后都有"上古秘辛"
   - 重要物品出场时，先展现神秘现象，再逐步揭示来历
   - 用"古老传说"、"宿命"、"因果"等词汇增加厚重感
   - 例：玉佩激活时，不要直接说"它帮他疗伤"，而是写"那枚从襁褓中便伴随他的玉佩，此刻泛起微弱青光，宛如远古星辰的低语..."

2. **世界观宏大感**
   - 始终暗示"眼前世界只是更大格局的碎片"
   - 用"万年前"、"上古时代"、"大道崩碎"等时间跨度词汇
   - 地名要有历史感：荒古遗迹、玄黄界、衍道帝尊
   - 重要事件要有"宿命感"：主角的觉醒不是偶然，而是"命中注定"

3. **伏笔铺设**
   - 重要信息"点到即止"，留给读者猜测空间
   - 每个配角都可能藏着秘密身份
   - 物品、地点、人物要有"多重身份"的可能
   - 例：写一个遗迹守护者，不要一次性交代全部背景，而是"他的眼神深处，似乎藏着某种难以言说的沧桑..."

4. **战斗场面的气势**
   - 用宏大的比喻："那道剑气划破天际，宛如天河倒悬"
   - 强调环境破坏感："方圆十里的山脉，在这一击之下化为平地"
   - 战斗要有"境界压制"的概念，高级别碾压低级别要有画面感

### 西红柿风格要素

我吃西红柿的核心优势在于**爽快节奏**与**清晰升级体系**：

1. **爽点节奏**
   - 每1-2章必须有一个小爽点（战斗胜利、获得资源、境界提升）
   - 每10-15章一个大爽点（收服灵宠、击败强敌、重大突破）
   - 爽点要"干净利落"，不拖泥带水
   - 例：击败敌人后，立即写收获："这一战，林衍不仅获得三枚兽核，更突破至凡境巅峰..."

2. **境界体系清晰**
   - 每次突破要明确写出：旧境界→新境界，以及具体变化
   - 用数字或具体描述让读者有概念："灵气淬体九次，便可尝试突破灵境"
   - 突破要有"门槛感"：不是轻松突破，而是有困难、有考验

3. **资源获取爽感**
   - 获得功法、丹药、法宝时，要详细描述品阶和价值
   - 让读者有"主角又变强了"的直观感受
   - 例："《衍道诀》，地阶中品功法，修炼大成可突破至帝境..."

4. **主角性格爽感**
   - "有仇必报，护短，不圣母"
   - 对敌人要果断，对伙伴要忠诚
   - 遇到挑衅，不忍气吞声，要有反击
   - 例："你抢我朋友的资源，今日我便让你十倍奉还！"

### 融合技巧

**战斗场景：西红柿式热血 + 辰东式气势**
```
"林衍手中剑芒暴涨，衍道本源之力涌入，剑气如星辰坠落，轰然斩下！
赵虎怒吼一声，荒古肉身诀催动至极限，拳劲如山岳撞击。
两道力量在半空碰撞，方圆百米的地面瞬间龟裂，尘土飞扬中，林衍的身影已然冲出，手中多了一枚闪烁微光的兽核——那头赤焰兽，已然毙命。"
```

**境界突破：西红柿式清晰 + 辰东式感悟**
```
"灵气九次淬体，林衍感到体内某种桎梏松动。
这是凡境的极限，也是灵境的门槛。
他闭目凝神，衍道本源在体内流转，那枚玉佩隐隐发热，仿佛远古的帝尊在低语——'破境，是大道的起步，而非终点...'
轰！体内一声轻响，灵气冲破屏障，林衍睁开双眼，眸中闪过一丝锐利——灵境初期，终于踏入！"
```

**场景铺垫：辰东式悬疑 + 西红柿式推进**
```
"这座遗迹的入口，被浓重的雾气笼罩。
'小心，'楚瑶低声道，'上古遗迹往往有守护者...'
林衍点头，玉佩已悄然发热——它在预警。
雾气深处，隐约传来低沉的呼吸声，那呼吸沉重如远古巨兽沉眠，每一次呼气，都震颤着四周的岩石...
'有守护兽，'林衍握紧长剑，'但这也是我们获取功法的机会。'"
```

## 章节结构规范

### 标准章节框架（约2000字/章）

1. **开篇承接**（约200字）
   - 与上一章的衔接或场景转换
   - 若是战斗中，快速进入状态
   - 若是新场景，用2-3句环境描写建立氛围

2. **主体推进**（约1500字）
   - **第一节奏点**：事件开始/战斗开场/探索启动
   - **第二节奏点**：危机出现/敌人强压/困境显现
   - **第三节奏点**：主角应对/反击/突破或解决
   - **爽点落实**：收获明确/境界提升/资源到手

3. **结尾铺垫**（约300字）
   - 留一个悬念或新线索
   - 为下一章制造期待感
   - 例："林衍握着新获得的源核碎片，玉佩再次泛起微光——它在指引他，下一处遗迹，在东方..."

### 节奏把控原则

- **小爽点间隔**：每2-4章一个（战斗胜利、获得物品、小突破）
- **大爽点间隔**：每卷设置5-6个（收服灵宠、击败Boss、境界大突破）
- **悬念铺设**：每章结尾留一个，每卷结尾留一个"大悬念"
- **避免拖沓**：战斗不超过连续5章，探索不超过连续3章

## 人物塑造要点

### 主角：林衍

**核心性格**：
- 坚韧但不固执
- 护短，有仇必报
- 有道心，会思考"力量的意义"
- 小腹黑，会用计谋

**写作要点**：
- 逆境时展现坚韧，不抱怨不放弃
- 面对敌人挑衅，反击要果断干净
- 面对伙伴遇险，要有"宁可自己受伤也要护住"的表现
- 成长中逐渐有"为众生而战"的意识萌芽

**对话风格**：
- 不啰嗦，话语简洁有力
- 对敌人："你找死。"（不废话）
- 对伙伴："一起走。"（信任感）
- 对强者："前辈，请教。"（有礼但不卑）

### 配角塑造要点

**楚瑶**：温柔但不软弱，说话要有"治愈感"但关键时刻果断
**赵虎**：豪爽热血，说话直接，战斗中怒吼增加气势
**凌月**：傲娇但靠谱，关键时刻展现能力
**墨尘**：冷峻神秘，话少但每句话有信息量

## 禁止事项

1. **禁止圣母剧情**：主角不会为了"救人"而让自己陷入必死困境
2. **禁止无脑反派**：反派要有目的、有算计，不是纯粹的恶
3. **禁止重复剧情**：每个地图的探索要有新元素，不重复套路
4. **禁止注水**：战斗描写不超过必要篇幅，不拖沓
5. **禁止金手指滥用**：衍道本源要"逐步解锁"，不是无限使用

## 幽默元素运用

适度幽默可以让小说更生动，调节紧张的剧情节奏。

### 幽默运用原则

1. **不影响主线严肃性**：幽默是点缀，不是主线基调
2. **不破坏人物性格**：幽默要符合角色设定，不能让严肃角色突然变得滑稽
3. **适度控制频率**：每5-10章可有一次幽默段落，不要每章都有
4. **避免低俗幽默**：幽默要有格调，不使用低俗、恶俗的笑话

### 幽默来源

#### 1. 主角的小腹黑

林衍性格中有"小腹黑"元素，可以用幽默的方式展现：

```
"那位长老要我交出功法？"
林衍嘴角微微上扬，"好的，我交。"
他从怀里掏出一本破旧的小册子——《基础养生指南》。
"这是我从小修炼的功法，请长老笑纳。"
长老的脸色，瞬间变得精彩纷呈。
```

**要点**：
- 幽默要有"反击"意味，不是单纯的搞笑
- 主角用"表面配合+实际坑人"的方式展现腹黑
- 不要让主角显得太轻浮

#### 2. 配角互动的幽默

伙伴之间的对话可以有轻松幽默：

```
赵虎："林兄，这异兽的肉能吃吗？"
林衍："能，但吃完你会后悔。"
赵虎："为什么？"
林衍："因为它辣。"
楚瑶在一旁轻笑："赵虎的肠胃，承受不住这'辣'的考验。"
赵虎："......我已经吃完了。"
三秒后，赵虎的脸变成了赤焰兽的颜色。
```

**要点**：
- 赵虎是"被调侃对象"，符合他豪爽但不细心的性格
- 楚瑶的"治愈感"笑点，温柔但有内容
- 不影响战斗/剧情主线，只是短暂调节气氛

#### 3. 灵宠/器灵的幽默性格

**赤焰兽（灵宠）**：
- 忠诚但有"小脾气"
- 被打败时会有"不服气"的表现

```
赤焰兽低着头，尾巴不情愿地甩了两下。
它在表达不满——明明是它先发现那枚兽核的，却被林衍"收缴"了。
林衍："下次我给你留一半。"
赤焰兽的尾巴立刻甩得欢快起来。
```

**凌月（器灵）**：
- 前期傲娇，有"嘴硬心软"的幽默感

```
凌月："我才不是帮你，只是这本功法太脏了，我顺手清理一下。"
林衍："谢谢。"
凌月："......不用谢！我只是......算了！"
她的身影消失在玉佩中，但林衍能感觉到，玉佩的温度微微上升——那是她在"生气"。
```

#### 4. 场景幽默

某些场景可以有幽默描写，但要点到为止：

```
遗迹的守护兽缓缓睁开眼，发出低沉的咆哮。
林衍握紧剑，准备战斗。
守护兽盯着他看了三秒，然后——打了个哈欠，继续睡了。
林衍："......它在无视我？"
楚瑶："可能它觉得......你不够强。"
林衍的剑差点掉在地上。
```

### 幽默禁区

1. **战斗高潮不用幽默**：紧张战斗中不要插入搞笑，会破坏气氛
2. **悲壮场景不用幽默**：配角牺牲、重要转折等场景不能幽默
3. **反派不能太滑稽**：反派可以有自己的幽默感（冷幽默），但不能变成"搞笑角色"
4. **不用网络梗**：避免使用"震惊"、"无语"等网络用语，保持玄幻小说的格调

## "十星降临"开篇写作指导

### 开篇设定

**时间**：2060年某夜凌晨0点

**背景**：人类科技发达，体质大幅提升，社会繁荣有序

**事件**：
1. 天空突然出现十颗奇异光点
2. 光点迅速扩张，温度骤升至200°C+
3. 全球水分蒸发，水蒸气汇聚某处形成巨大"云柱"
4. 灾难持续数小时后，云柱崩散，灵气爆发

### 开篇写作要点

**第一章必须包含的元素**（约2000字）：

1. **时代感**：
   - 用几句描写展示2060年的科技生活（智能设备、强化体质、月球基地等）
   - 主角的"普通"——不是"强化人"阶层，体质中等
   - 让读者感受到"人类很强大"的自信

2. **异象降临**：
   - 十颗光点的描写要有"神秘感"和"画面冲击力"
   - 温度骤升的压迫感——皮肤灼烧、呼吸困难
   - 水蒸气汇聚的壮观——云柱直通天际

3. **主角的绝境**：
   - 高温中艰难求生，身边有人倒下
   - 玉佩的第一次反应——微微发热，提供一丝保护
   - 不要解释玉佩来历，只展现现象

4. **悬念铺设**：
   - 十星是什么？（不回答，只提问）
   - 为什么水蒸气汇聚？（暗示有力量牵引）
   - 玉佩为什么有反应？（不解释）

### 开篇风格融合示例

**描写十星降临**：
```
"凌晨零点整。

林衍刚关上智能终端，准备入睡。窗外，城市的霓虹依旧闪烁，2060年的繁华一如既往。

忽然，一道光芒刺入他的眼帘。

他抬头望去——天空中，十颗奇异的星点同时出现。

它们不是普通的星辰，而是某种...更古老的存在。光芒从微小的一点开始，却以惊人的速度扩张，仿佛某种封印正在被撕裂。

温度，开始升高。

先是燥热，然后是灼烧感。林衍感到皮肤像被火焰舔舐，呼吸变得艰难——空气中的水分正在消失，以肉眼可见的速度消失。

窗外，海洋的方向，一道巨大的云柱正在形成。水蒸气，全球所有的水蒸气，似乎都在向那里汇聚...

那云柱直通天际，宛如上古神话中的'通天之柱'。

林衍的意识开始模糊，高温正在吞噬他的一切。

就在他即将倒下的瞬间，胸口那枚从小佩戴的玉佩，忽然微微发热。

那热度不是灼烧，而是...某种守护。仿佛远古的低语在耳边响起——

'活下去...大道，正在觉醒...'"
```

### 开篇禁区

1. **不要过度解释**：十星的来历、玉佩的来历、水蒸气汇聚的原因——全部留悬念
2. **不要让主角表现太强**：他只是勉强存活，不是主动战斗
3. **不要写太多配角**：前10章只写主角和主角视角看到的灾难
4. **不要科技细节过多**：只需几句展示时代，然后灾难打破一切

## 参考资料使用

创作时需要查阅以下reference文件：

- `references/world-settings.md`：世界观设定、修炼体系、核心概念
- `references/outline.md`：四卷大纲、关键节点、爽点设计
- `references/characters.md`：主角、配角、反派详细设定

**查阅时机**：
- 开始新卷时：阅读本卷大纲和关键节点
- 涉及新角色时：查阅对应角色设定
- 涉及境界/功法/物品时：查阅世界观设定中的品阶体系
- 需要设计战斗/突破场景时：查阅修炼体系

## 写作流程

当用户请求创作时，按以下流程执行：

1. **确认创作位置**
   - 当前是第几卷、第几章？
   - 根据outline.md确认本章要达成的目标

2. **查阅必要资料**
   - 本章涉及的场景、角色、物品的设定
   - 本卷的关键节点和爽点安排

3. **构建章节框架**
   - 确定本章的爽点类型（战斗胜利/获得资源/境界突破/剧情推进）
   - 确定结尾悬念

4. **正式创作**
   - 按章节结构规范展开
   - 融合两种风格要素

5. **创作后检查**
   - 是否有爽点？
   - 是否有悬念/铺垫？
   - 人物性格是否一致？
   - 境界/设定是否准确？
