# 《裂星衍道》写作指导

本skill用于创作百万字玄幻小说《裂星衍道》，融合辰东与我吃西红柿两位顶级网文作家的风格精髓。

## 写作风格融合指南

### 辰东风格要素

辰东的核心优势在于**宏大世界观构建**与**悬疑铺垫能力**：

1. **神秘感营造**
   - 每个遗迹、神器背后都有"上古秘辛"
   - 重要物品出场时，先展现神秘现象，再逐步揭示来历
   - 用"古老传说"、"宿命"、"因果"等词汇增加厚重感
   - 例：玉佩激活时，不要直接说"它帮他疗伤"，而是写"那枚从襁褓中便伴随他的玉佩，此刻泛起微弱青光，宛如远古星辰的低语..."

2. **世界观宏大感**
   - 始终暗示"眼前世界只是更大格局的碎片"
   - 用"万年前"、"上古时代"、"大道崩碎"等时间跨度词汇
   - 地名要有历史感：荒古遗迹、玄黄界、衍道帝尊
   - 重要事件要有"宿命感"：主角的觉醒不是偶然，而是"命中注定"

3. **伏笔铺设**
   - 重要信息"点到即止"，留给读者猜测空间
   - 每个配角都可能藏着秘密身份
   - 物品、地点、人物要有"多重身份"的可能
   - 例：写一个遗迹守护者，不要一次性交代全部背景，而是"他的眼神深处，似乎藏着某种难以言说的沧桑..."

4. **战斗场面的气势**
   - 用宏大的比喻："那道剑气划破天际，宛如天河倒悬"
   - 强调环境破坏感："方圆十里的山脉，在这一击之下化为平地"
   - 战斗要有"境界压制"的概念，高级别碾压低级别要有画面感

### 西红柿风格要素

我吃西红柿的核心优势在于**爽快节奏**与**清晰升级体系**：

1. **爽点节奏**
   - 每1-2章必须有一个小爽点（战斗胜利、获得资源、境界提升）
   - 每10-15章一个大爽点（收服灵宠、击败强敌、重大突破）
   - 爽点要"干净利落"，不拖泥带水
   - 例：击败敌人后，立即写收获："这一战，林衍不仅获得三枚兽核，更突破至凡境巅峰..."

2. **境界体系清晰**
   - 每次突破要明确写出：旧境界→新境界，以及具体变化
   - 用数字或具体描述让读者有概念："灵气淬体九次，便可尝试突破灵境"
   - 突破要有"门槛感"：不是轻松突破，而是有困难、有考验

3. **资源获取爽感**
   - 获得功法、丹药、法宝时，要详细描述品阶和价值
   - 让读者有"主角又变强了"的直观感受
   - 例："《衍道诀》，地阶中品功法，修炼大成可突破至帝境..."

4. **主角性格爽感**
   - "有仇必报，护短，不圣母"
   - 对敌人要果断，对伙伴要忠诚
   - 遇到挑衅，不忍气吞声，要有反击
   - 例："你抢我朋友的资源，今日我便让你十倍奉还！"

### 融合技巧

**战斗场景：西红柿式热血 + 辰东式气势**
```
"林衍手中剑芒暴涨，衍道本源之力涌入，剑气如星辰坠落，轰然斩下！
赵虎怒吼一声，荒古肉身诀催动至极限，拳劲如山岳撞击。
两道力量在半空碰撞，方圆百米的地面瞬间龟裂，尘土飞扬中，林衍的身影已然冲出，手中多了一枚闪烁微光的兽核——那头赤焰兽，已然毙命。"
```

**境界突破：西红柿式清晰 + 辰东式感悟**
```
"灵气九次淬体，林衍感到体内某种桎梏松动。
这是凡境的极限，也是灵境的门槛。
他闭目凝神，衍道本源在体内流转，那枚玉佩隐隐发热，仿佛远古的帝尊在低语——'破境，是大道的起步，而非终点...'
轰！体内一声轻响，灵气冲破屏障，林衍睁开双眼，眸中闪过一丝锐利——灵境初期，终于踏入！"
```

**场景铺垫：辰东式悬疑 + 西红柿式推进**
```
"这座遗迹的入口，被浓重的雾气笼罩。
'小心，'楚瑶低声道，'上古遗迹往往有守护者...'
林衍点头，玉佩已悄然发热——它在预警。
雾气深处，隐约传来低沉的呼吸声，那呼吸沉重如远古巨兽沉眠，每一次呼气，都震颤着四周的岩石...
'有守护兽，'林衍握紧长剑，'但这也是我们获取功法的机会。'"
```

## 章节结构规范

### 字数要求

**每章正文字数范围：2000-2500字**

字数统计方法：
- **只统计正文内容**
- 不含标题行（如"第一卷 第1章 2060年的夜"）
- 不含末尾"---"后的【本章要点】总结部分
- 包含对话、描写、心理活动、环境描写

**字数检查流程**：
创作完成后必须检查字数，不达标则自动修改：
1. 统计正文字数
2. 如果 < 2000字 → 扩充内容，最多修改2轮
3. 如果 > 2500字 → 精简内容，最多修改2轮
4. 达标后继续后续流程

**扩充方向**（字数不足时）：
- 环境描写：增加场景细节、氛围渲染
- 心理活动：增加主角的内心思考、情感波动
- 对话互动：增加角色间的对话，推进剧情
- 动作细节：增加战斗或动作的详细描写

### 标准章节框架（约2000字/章）

1. **开篇承接**（约200-300字）
   - 与上一章的衔接或场景转换
   - 若是战斗中，快速进入状态
   - 若是新场景，用3-5句环境描写建立氛围
   - 可加入主角的心理活动或回忆片段

2. **主体推进**（约1400-1600字）
   - **第一节奏点**：事件开始/战斗开场/探索启动（约400字）
   - **第二节奏点**：危机出现/敌人强压/困境显现（约400字）
   - **第三节奏点**：主角应对/反击/突破或解决（约400字）
   - **爽点落实**：收获明确/境界提升/资源到手（约200字）
   - 注意：每个节奏点要有足够的细节描写，不要过于简略

3. **结尾铺垫**（约300-400字）
   - 留一个悬念或新线索
   - 为下一章制造期待感
   - 可加入主角的思考或预感
   - 例："林衍握着新获得的源核碎片，玉佩再次泛起微光——它在指引他，下一处遗迹，在东方..."

### 扩充技巧

当字数不足时，优先扩充以下内容：

1. **环境描写**：
   - 天气、光线、声音、气味
   - 建筑物、地形、植被的细节
   - 时间流逝的感觉
   - 例：不要只写"天气很热"，而是"烈日炙烤着大地，空气中弥漫着干燥的热浪，连远处的山脉都笼罩在一层朦胧的尘埃中..."

2. **心理活动**：
   - 主角的思考过程
   - 对现状的分析
   - 内心的矛盾与挣扎
   - 对未来的担忧或期待
   - 例：不要只写"林衍很担心"，而是"林衍的心中涌起一股不安——这高温来得太突然，太诡异。科学？天灾？还是某种更神秘的力量？他的脑海中闪过无数猜测，却找不到一个合理的答案..."

3. **动作细节**：
   - 每一个动作的具体描写
   - 身体的感觉变化
   - 力量的流动
   - 例：不要只写"林衍站起来"，而是"林衍咬紧牙关，用尽全身力气撑起身体。高温让他的肌肉几乎失去知觉，但求生本能驱使着他——必须站起来，必须活下去..."

4. **对话互动**：
   - 增加对话回合
   - 加入语气、表情的描写
   - 对话中的思考停顿
   - 例：不要只写"林衍问：这是什么？"，而是"林衍的声音颤抖着，'这......这是什么？'他的喉咙因高温而干涩，每一个字都像是用沙砾摩擦出来的..."

### 节奏把控原则

- **小爽点间隔**：每2-4章一个（战斗胜利、获得物品、小突破）
- **大爽点间隔**：每卷设置5-6个（收服灵宠、击败Boss、境界大突破）
- **悬念铺设**：每章结尾留一个，每卷结尾留一个"大悬念"
- **避免拖沓**：战斗不超过连续5章，探索不超过连续3章

## 人物塑造要点

### 主角：林衍

**核心性格**：
- 坚韧但不固执
- 护短，有仇必报
- 有道心，会思考"力量的意义"
- 小腹黑，会用计谋

**写作要点**：
- 逆境时展现坚韧，不抱怨不放弃
- 面对敌人挑衅，反击要果断干净
- 面对伙伴遇险，要有"宁可自己受伤也要护住"的表现
- 成长中逐渐有"为众生而战"的意识萌芽

**对话风格**：
- 不啰嗦，话语简洁有力
- 对敌人："你找死。"（不废话）
- 对伙伴："一起走。"（信任感）
- 对强者："前辈，请教。"（有礼但不卑）

### 配角塑造要点

**楚瑶**：温柔但不软弱，说话要有"治愈感"但关键时刻果断
**赵虎**：豪爽热血，说话直接，战斗中怒吼增加气势
**凌月**：傲娇但靠谱，关键时刻展现能力
**墨尘**：冷峻神秘，话少但每句话有信息量

## 禁止事项

1. **禁止圣母剧情**：主角不会为了"救人"而让自己陷入必死困境
2. **禁止无脑反派**：反派要有目的、有算计，不是纯粹的恶
3. **禁止重复剧情**：每个地图的探索要有新元素，不重复套路
4. **禁止注水**：战斗描写不超过必要篇幅，不拖沓
5. **禁止金手指滥用**：衍道本源要"逐步解锁"，不是无限使用

## 幽默元素运用

幽默是调节紧张节奏的重要元素，但必须**自然融入**，不能生硬插入。

### 幽默融入流程

在**第二阶段：构建章节框架**时，需要判断：

**是否适合加入幽默？**
- ✅ 适合：休整/准备场景、团队日常互动、收获分配、轻松对话
- ❌ 不适合：战斗高潮、危机紧迫、悲壮场景、敌人对峙

**适合时，选择幽默类型：**
根据本章场景选择一种幽默类型，不要堆砌。

### 幽默类型与场景匹配

| 场景类型 | 推荐幽默类型 | 示例 |
|----------|--------------|------|
| 休整/准备 | 赵虎被调侃 | 赵虎吃东西后悔、做事鲁莽被提醒 |
| 收获分配 | 主角腹黑坑人 | 分配资源时故意坑贪婪者 |
| 团队日常 | 楚瑶温柔调侃 | 用温柔语气说出扎心的话 |
| 探索发现 | 场景小幽默 | 发现意想不到的情况 |
| 灵宠互动 | 灵宠小脾气 | （灵宠出现后） |

### 幽默写作要点

**1. 赵虎被调侃**（最常见的幽默来源）

赵虎性格豪爽、做事不细心，适合做"被调侃对象"：

```
赵虎看着手里那块兽肉，咽了咽口水："这能烤吗？"
林衍："能，但吃完你会后悔。"
赵虎已经咬下去了："为什么？"
楚瑶轻声道："上次吃完，赵虎躺了三天。"
赵虎的脸色变了："......我好像记得这事。"
林衍："记得？那你为什么还吃？"
赵虎："......它太香了。"
```

**要点**：
- 赵虎自己"作死"，然后后悔
- 林衍/楚瑶只是旁观调侃，不阻止
- 不要让赵虎显得愚蠢，只是"豪爽不计后果"

**2. 主角腹黑坑人**

林衍的"腹黑"用于对付贪婪者/敌人，不是坑朋友：

```
那手下盯着林衍手里的兽核："林公子，能否......"
林衍嘴角微扬："可以，我给你。"
他从玉佩里取出——一枚凡境初期兽核。
"这是我最早获得的兽核，送你。"
那手下的脸色，精彩纷呈。
```

**要点**：
- 对方贪婪→林衍表面答应→给出最差的东西
- 有反击意味，不是单纯搞笑
- 保持主角的从容，不要显得轻浮

**3. 楚瑶温柔调侃**

楚瑶说话温柔，但内容可以扎心：

```
赵虎："我这次肯定不会躺三天！"
楚瑶轻声："嗯，上次是三天。这次可能更久。"
赵虎："......楚瑶妹子，你这话太扎心了。"
楚瑶微笑："我只是陈述事实。"
```

**要点**：
- 语气温柔，内容直接
- 不带恶意，是朋友间的提醒
- 用微笑/轻声增加"治愈感"

**4. 场景小幽默**

发现意外情况时的轻松反应：

```
林衍推开遗迹石门——空的。
赵虎："......就这样？"
楚瑶："可能我们来晚了。"
林衍："来晚了？玉佩刚才还在发热。"
三人对视。
林衍："......玉佩，你坑我？"
玉佩没有回应。
```

**要点**：
- 情况出乎意料→短暂无语→快速接受
- 不影响主线，只是调节气氛
- 点到为止，不要展开成搞笑段落

### 幽默禁区

1. **战斗高潮**：紧张战斗中绝不插入幽默
2. **危机紧迫**：被追杀、被困时不幽默
3. **敌人对峙**：与敌人谈判/对抗时不幽默
4. **悲壮场景**：牺牲、重伤、重要转折不幽默
5. **不用网络梗**：不用"震惊"、"无语"等网络用语

### 幽默频率控制

- **每5-8章可有一次幽默段落**（约50-100字）
- **不要每章都有**，会稀释严肃基调
- **一次只用一种幽默类型**，不要堆砌

## "十星降临"开篇写作指导

### 开篇设定

**时间**：2060年某夜凌晨0点

**背景**：人类科技发达，体质大幅提升，社会繁荣有序

**事件**：
1. 天空突然出现十颗奇异光点
2. 光点迅速扩张，温度骤升至200°C+
3. 全球水分蒸发，水蒸气汇聚某处形成巨大"云柱"
4. 灾难持续数小时后，云柱崩散，灵气爆发

### 开篇写作要点

**第一章必须包含的元素**（约2000字）：

1. **时代感**：
   - 用几句描写展示2060年的科技生活（智能设备、强化体质、月球基地等）
   - 主角的"普通"——不是"强化人"阶层，体质中等
   - 让读者感受到"人类很强大"的自信

2. **异象降临**：
   - 十颗光点的描写要有"神秘感"和"画面冲击力"
   - 温度骤升的压迫感——皮肤灼烧、呼吸困难
   - 水蒸气汇聚的壮观——云柱直通天际

3. **主角的绝境**：
   - 高温中艰难求生，身边有人倒下
   - 玉佩的第一次反应——微微发热，提供一丝保护
   - 不要解释玉佩来历，只展现现象

4. **悬念铺设**：
   - 十星是什么？（不回答，只提问）
   - 为什么水蒸气汇聚？（暗示有力量牵引）
   - 玉佩为什么有反应？（不解释）

### 开篇风格融合示例

**描写十星降临**：
```
"凌晨零点整。

林衍刚关上智能终端，准备入睡。窗外，城市的霓虹依旧闪烁，2060年的繁华一如既往。

忽然，一道光芒刺入他的眼帘。

他抬头望去——天空中，十颗奇异的星点同时出现。

它们不是普通的星辰，而是某种...更古老的存在。光芒从微小的一点开始，却以惊人的速度扩张，仿佛某种封印正在被撕裂。

温度，开始升高。

先是燥热，然后是灼烧感。林衍感到皮肤像被火焰舔舐，呼吸变得艰难——空气中的水分正在消失，以肉眼可见的速度消失。

窗外，海洋的方向，一道巨大的云柱正在形成。水蒸气，全球所有的水蒸气，似乎都在向那里汇聚...

那云柱直通天际，宛如上古神话中的'通天之柱'。

林衍的意识开始模糊，高温正在吞噬他的一切。

就在他即将倒下的瞬间，胸口那枚从小佩戴的玉佩，忽然微微发热。

那热度不是灼烧，而是...某种守护。仿佛远古的低语在耳边响起——

'活下去...大道，正在觉醒...'"
```

### 开篇禁区

1. **不要过度解释**：十星的来历、玉佩的来历、水蒸气汇聚的原因——全部留悬念
2. **不要让主角表现太强**：他只是勉强存活，不是主动战斗
3. **不要写太多配角**：前10章只写主角和主角视角看到的灾难
4. **不要科技细节过多**：只需几句展示时代，然后灾难打破一切

## 写作流程（状态注入版）

当用户请求创作时，按以下流程执行：

### 第一阶段：状态注入（必须执行）

1. **读取状态文件**
   - 读取 `references/writing-snapshot.md`（获取当前场景、情绪、悬念）
   - 读取 `references/writing-style-guide.md`（获取风格去AI指导）
   - 读取 `references/chapter-summary.md`（获取长期上下文）

2. **状态确认**
   - 在内部确认关键状态信息已读取
   - 如状态文件不存在或过旧，提醒用户需要先创建/更新

### 第二阶段：构建框架

3. **查阅其他必要资料**
   - 根据本章内容查阅：world-settings、characters、foreshadowing-tracker
   - 检查是否有伏笔可以铺垫或揭示

4. **构建章节框架**
   - 确定本章爽点类型
   - 确定结尾悬念
   - 标记描写密度（高密度场景vs低密度场景）
   - 确定是否有伏笔需要处理
   
5. **判断幽默时机**（关键步骤）
   - 判断本章场景类型：休整/战斗/探索/日常/危机？
   - 休整/准备/日常场景 → 可加入幽默
   - 战斗/危机/敌人对峙 → 不加幽默
   - 若适合幽默 → 选择一种幽默类型（赵虎被调侃/主角腹黑/楚瑶调侃/场景小幽默）
   - 标记幽默位置：在哪个节奏点插入（通常是第一节奏点或结尾铺垫）

### 第三阶段：正式创作

6. **创作章节正文**
   - 按章节结构规范展开（2000-2500字）
   - **应用风格去AI指导**（对话口语化、描写节奏、心理真实感、转折自然化）
   - 融合辰东+西红柿风格要素
   - **若标记了幽默 → 在指定位置自然插入**（约50-100字）

7. **字数验证**（必须执行）
   - 统计正文字数（不含标题和【本章要点】）
   - 如不达标，扩充/精简（最多2轮）

### 第四阶段：输出与更新

7. **输出章节**
   - 输出正文内容
   - **附带状态验证声明**
   - 附带风格验证声明
   - 附带【本章要点】总结

8. **更新状态文件**（必须执行）
   - 更新 `references/writing-snapshot.md`（新的结束状态）
   - 如有境界突破/团队变化，更新 `references/chapter-summary.md`
   - 如有新伏笔，更新 `references/foreshadowing-tracker.md`

## 伏笔管理

### 伏笔的重要性

伏笔是辰东风格的核心要素之一。好的伏笔能让读者产生期待感，并在揭示时获得满足感。

### 伏笔类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 世界悬念 | 关于世界真相的伏笔 | 十星的来历、地球是玄黄界碎片 |
| 身世悬念 | 关于主角身世的伏笔 | 玉佩来历、主角是帝尊转世 |
| 物品悬念 | 关于重要物品的伏笔 | 玉佩的功能、功法的秘密 |
| 势力悬念 | 关于势力真相的伏笔 | 隐世家族的目的、幕后黑手 |
| 伙伴悬念 | 关于角色身份的伏笔 | 楚瑶的来历、墨尘的身世 |

### 伏笔埋下原则

1. **点到即止**：不要一次性透露太多信息，留有猜测空间
2. **多次铺垫**：重要伏笔可以在多个章节反复暗示
3. **合理时机**：伏笔要埋在合适的场景，不能生硬插入
4. **记录追踪**：每埋下一个伏笔，必须立即记录到追踪文件

### 伏笔揭示原则

1. **预期揭示**：按照追踪文件中的预期揭示位置安排
2. **震撼感**：揭示要有画面感和情感冲击
3. **逻辑闭环**：揭示内容要与埋下时的设定一致
4. **及时更新**：揭示后立即更新伏笔追踪文件的状态

### 伏笔追踪文件使用

文件位置：`references/foreshadowing-tracker.md`

**埋下新伏笔时**：
```
| VP-XXX | 第X章 | [伏笔简述] | [预期揭示位置] | - | 已埋下 |
```

**揭示伏笔时**：
1. 更新表格中的"揭示章节"和"状态"
2. 添加揭示详情：
```
### VP-XXX 揭示详情
**揭示章节**：第X章
**揭示内容**：[具体揭示的内容]
**揭示方式**：[通过什么方式揭示]
```
