# 防止AI化写作指导

本skill用于在创作和润色章节时自动检查并去除AI化写作痕迹，确保文字自然流畅，符合真正小说的阅读体验。

**重要：每次创作新章节或润色章节时，必须自动执行本指导。**

---

## AI化写作的核心问题

### 问题类型识别

1. **对话机械化**
   - 固定"语气——XXX"标签（如"声音豪爽，目光中带着XXX"）
   - 缺乏真实口语表达
   - 对话回合过于规整

2. **描写数据化**
   - 大量"---"分隔符作为段落过渡
   - 数值化描述过度（"约两米"、"约五倍"、"约三个小时"）
   - 缺乏氛围描写和感官描写

3. **心理系统化**
   - 末尾"本章要点"总结表格
   - 心理活动过于条理化（"林衍理解了XXX......他喃喃道......")
   - 缺乏碎片化、跳跃性思考

4. **转折模板化**
   - 固定"忽然——"、"突然——"开头
   - 固定"约X秒后——"、"约X分钟后——"过渡
   - 缺乏自然铺垫和节奏变化

5. **句式重复**
   - "XXX......YYY......"高频重复
   - "XXX——YYY"固定格式过多
   - 缺乏长短句交替、倒装句、感叹句

---

## 五项去AI化标准

### 1. 对话口语化

**禁止格式**：
- `XXX的声音[形容词]，目光中带着[形容词]。"对话内容"`
- `XXX的语气——[形容词]。"对话内容"`

**正确格式**：
```
赵虎咧嘴一笑："林兄，这异兽肉太香了，我忍不住就吃了。"

"你上次吃完躺了三天。"林衍看着他那狼吞虎咽的样子。

"......这次不会！"
```

**技巧**：
- 增加语气词（"嗯"、"啊"、"嘛"、"咯"、"吧"）
- 增加短句、中断、口语表达
- 描写动作配合对话，而非固定标签
- 让对话有节奏感、真实感

---

### 2. 描写文学化

**禁止格式**：
- 用"---"作为段落过渡
- 纯数据化描述："体型扩大了约两倍"

**正确格式**：
```
那异兽体型接近中型犬，血红色的毛发在昏暗中闪烁如凝固的血块，金色的瞳孔锐利如刀，锁定林衍的位置。
```

**技巧**：
- 用氛围描写替代分隔符："夜幕逐渐深沉，灵气逐渐活跃"
- 用感官描写替代数据：视觉、听觉、嗅觉、触觉
- 用比喻修辞增加画面感："如"、"仿佛"、"宛如"
- 具体细节替代笼统数值

---

### 3. 心理真实化

**禁止格式**：
- 末尾【本章要点】表格
- "林衍理解了XXX......他喃喃道......"条理化心理

**正确格式**：
```
林衍的心跳不由加快。这异兽......比之前遇到的更强。如果正面战斗......

他握紧金属杆。不行，不能退。

必须战斗。必须活下去。
```

**技巧**：
- 删除末尾表格总结
- 心理活动碎片化、跳跃性
- 增加情感波动、矛盾思考
- 用短句、中断表现心理节奏

---

### 4. 转折自然化

**禁止格式**：
- "忽然——XXX"
- "约X分钟后——XXX"
- "突然，XXX"

**正确格式**：
```
废墟的深处，隐约传来低沉的呼吸。

林衍的心跳加速。有异兽？

他握紧金属杆，目光扫向阴影深处。
```

**技巧**：
- 用氛围铺垫替代固定转折词
- 用节奏变化替代时间数值
- 用感官触发替代"忽然"
- 让转折有铺垫、有预期

---

### 5. 句式多样化

**禁止格式**：
- 全章"XXX......YYY......"超过10处
- "XXX——YYY"固定格式超过5处

**正确格式**：
- 长短句交替
- 倒装句、感叹句、疑问句穿插
- 描述句、动作句、对话句混合

**技巧**：
- 检查每章"......"使用次数，控制在5处以内
- 检查"——"使用次数，控制在5处以内
- 增加感叹句："不可能！"
- 增加疑问句："这是什么？"
- 增加倒装句："危险，正在靠近"

---

## 自动检查流程

### 创作完成后必须执行

1. **统计检查**：
   - "......"出现次数 ≤ 5处
   - "——"出现次数 ≤ 5处
   - "---"分隔符 ≤ 3处
   - "忽然——" ≤ 2处
   - "约X秒后——" ≤ 2处

2. **格式检查**：
   - 无末尾【本章要点】表格
   - 无固定对话标签格式
   - 无条理化心理描写

3. **质量检查**：
   - 每章约2000字
   - 有环境描写、心理活动、动作细节
   - 对话自然、转折流畅

### 不达标则修改

若检查不通过，必须修改直至达标：
- 最多修改2轮
- 修改后重新检查
- 记录修改原因

---

## 输出格式

创作完成后，输出以下验证声明：

```
【去AI化验证】
- 对话口语化：已应用，固定标签已去除
- 描写文学化：已应用，分隔符已减少至X处
- 心理真实化：已应用，末尾表格已删除
- 转折自然化：已应用，固定转折词已减少
- 句式多样化：已应用，"......"X处，"——"X处
- 字数验证：约XXX字，达标
```

## 常见问题修正示例

### 问题对话修正

**原AI化**：
```
赵虎的声音豪爽，目光中带着好奇。"我是赵虎，灵境初期修炼者。"
```

**修正后**：
```
"我是赵虎！"那人咧嘴一笑，拍了拍胸脯，"灵境初期，在灵域外围混口饭吃。"
```

### 问题描写修正

**原AI化**：
```
---
异兽体型扩大了约两倍。
---
```

**修正后**：
```
那异兽体型接近中型犬，血红色的毛发在昏暗中闪烁，金色的瞳孔锐利如刀。
```

### 问题心理修正

**原AI化**：
```
林衍理解了灵域的危险。灵域内有更强异兽，需要谨慎。他喃喃道："必须小心......"
```

**修正后**：
```
灵域内......更强异兽？

林衍握紧金属杆。不能大意。

这地方，比外围危险得多。
```

## 与主skill配合

本指导是 `liexing-xiandao-writing` skill的补充，必须在创作流程中自动执行：

1. **创作前**：阅读本指导，确认去AI化标准
2. **创作中**：应用五项标准，避免AI化格式
3. **创作后**：执行自动检查，输出验证声明

**不执行本指导视为创作不合格，必须修改。**
